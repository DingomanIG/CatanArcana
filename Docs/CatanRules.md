# 카탄의 개척자들 - 게임 규칙

## 게임 목표
자원을 모아 도로, 마을, 도시를 건설하여 **10 승리점**을 먼저 달성하면 승리!

## 승리점 획득 방법
| 항목 | 점수 |
|------|------|
| 마을 | 1점 |
| 도시 | 2점 |
| 최장교역로 (도로 5개 이상 연속) | 2점 |
| 최대기사단 (기사카드 3장 이상 사용) | 2점 |
| 승리점 발전카드 | 1점 |

## 자원 (5종)
| 자원 | 생산 타일 | 색상 |
|------|-----------|------|
| 목재 | 숲 | 초록 |
| 벽돌 | 언덕 | 주황 |
| 양모 | 초원 | 연두 |
| 밀 | 밭 | 노랑 |
| 광석 | 산 | 회색 |

사막 타일은 자원을 생산하지 않습니다.

## 턴 진행
### 1단계: 주사위 굴리기
- 2개의 주사위를 굴립니다 (2~12)
- 나온 숫자와 같은 숫자 토큰이 있는 타일에서 자원 생산
- 해당 타일에 마을이 인접하면 자원 1개, 도시면 2개 획득
- **7이 나오면**: 도적 이동 (아래 '도적' 참조)

### 2단계: 거래
- **은행 거래**: 같은 자원 4개 → 원하는 자원 1개
- **항구 거래**: 항구에 마을/도시가 있으면 유리한 비율로 거래
  - 일반 항구 (3:1): 같은 자원 3개 → 원하는 자원 1개
  - 특수 항구 (2:1): 해당 자원 2개 → 원하는 자원 1개
- **플레이어 거래**: 다른 플레이어와 자유롭게 자원 교환 제안 가능

### 3단계: 건설
자원을 사용하여 건물을 지을 수 있습니다.

| 건물 | 필요 자원 | 효과 |
|------|-----------|------|
| 도로 | 목재 1 + 벽돌 1 | 교역로 확장 |
| 마을 | 목재 1 + 벽돌 1 + 양모 1 + 밀 1 | 인접 타일에서 자원 1개 생산 (1VP) |
| 도시 | 밀 2 + 광석 3 | 마을을 업그레이드, 자원 2개 생산 (2VP) |
| 발전카드 | 양모 1 + 밀 1 + 광석 1 | 다양한 특수 효과 |

### 4단계: 턴 종료
건설과 거래를 마쳤으면 턴을 종료합니다.

## 건설 규칙
- **도로**: 자신의 도로, 마을, 도시에 연결된 변에만 건설 가능
- **마을**: 자신의 도로에 연결된 교차점에만 건설 가능. 다른 마을/도시와 최소 도로 2칸 이상 떨어져야 함 (거리 규칙)
- **도시**: 자신의 마을을 업그레이드하여 건설

## 초기 배치
1. 순서대로 각 플레이어가 마을 1개 + 도로 1개 배치
2. 역순으로 마을 1개 + 도로 1개 추가 배치
3. 두 번째 마을에 인접한 타일의 자원을 각 1개씩 획득

## 도적
- **7이 나오면**:
  1. 자원 8장 이상 보유한 플레이어는 절반(내림)을 버려야 합니다
  2. 주사위를 굴린 플레이어가 도적을 원하는 타일로 이동
  3. 도적이 있는 타일은 자원을 생산하지 않습니다
  4. 도적을 옮긴 타일에 마을/도시가 있는 플레이어 1명에게서 자원 1장을 무작위로 가져옵니다

## 발전카드
한 턴에 1장만 사용 가능. 구매한 턴에는 사용 불가.

| 카드 | 효과 |
|------|------|
| 기사 (14장) | 도적을 이동시킵니다. 최대기사단 달성에 기여 |
| 승리점 (5장) | 즉시 1점 획득 (비공개, 10점 달성 시 공개) |
| 도로건설 (2장) | 도로 2개를 무료로 건설 |
| 풍년 (2장) | 은행에서 원하는 자원 2개 획득 |
| 독점 (2장) | 자원 1종을 선언하면 모든 플레이어가 해당 자원을 전부 넘겨줌 |

## 특수 보너스
- **최장교역로**: 연속으로 이어진 도로가 5개 이상인 플레이어에게 2점. 다른 플레이어가 더 긴 도로를 만들면 이동
- **최대기사단**: 기사카드를 3장 이상 사용한 플레이어에게 2점. 다른 플레이어가 더 많이 사용하면 이동
