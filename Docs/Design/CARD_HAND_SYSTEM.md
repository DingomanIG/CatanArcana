# 카드 핸드 시스템 설계

## 개요
Balatro 스타일 부채꼴 카드 핸드를 두 개 운영하여 자원카드와 발전카드를 분리 관리한다.

---

## 핸드 구조

### 왼쪽 핸드 — 자원 카드
- **위치**: 화면 하단 왼쪽
- **카드 종류**: 목재, 벽돌, 양모, 밀, 광석 (5종)
- **정렬 규칙**: 자원 타입별 그룹핑, 고정 순서
  - `목재 → 벽돌 → 양모 → 밀 → 광석`
  - 같은 자원끼리 뭉쳐서 배치
- **인터랙션**:
  - 호버: 카드 위로 떠오름 + 확대 (자원 설명 표시)
  - 클릭: 단일 선택 (거래용)
  - 다중 선택: 도적 디스카드 시 활성화 (복수 카드 선택 가능)
  - 드래그: 거래 대상 지정 (추후)

### 오른쪽 핸드 — 발전 카드 + 보너스
- **위치**: 화면 하단 오른쪽
- **카드 종류**:
  - 발전카드: 기사, 승리점, 도로건설, 풍년, 독점
  - 보너스카드: 최장도로, 최강기사 (사용 불가, 표시 전용)
- **정렬 규칙**: 카드 타입별 그룹핑
  - `기사 → 도로건설 → 풍년 → 독점 → 승리점 → [최장도로] → [최강기사]`
  - 보너스카드는 항상 맨 오른쪽
- **인터랙션**:
  - 호버: 카드 위로 떠오름 + 확대 (카드 효과 설명 표시)
  - 드래그 위로: 카드 사용 시도
    - 사용 가능 → 카드 날아가며 효과 발동 --화면 중앙에 위치하면 사이즈가 커짐 스캐일 바운드되고 카드 발동
    - 사용 불가 → 좌우 흔들림 연출 후 핸드로 복귀
  - 보너스카드: 드래그 불가, 호버만 가능

---

## 카드 생성/제거 연출

### 자원 카드
- **획득**: 타일 월드 좌표 → 스크린 좌표 변환 → 왼쪽 핸드 목표 위치로 날아옴
- **소모**: 핸드에서 위로 날아가며 사라짐 (거래/건설 시)
- **디스카드**: 선택된 카드들이 화면 중앙(도적)으로 날아감

### 발전 카드
- **구매**: 덱 UI 위치 → 오른쪽 핸드 목표 위치로 날아옴
- **사용**: 드래그 후 상단 임계선 초과 시 → 위로 날아가며 효과 발동
- **사용 불가**: 좌우 흔들림(shake) → 원래 위치로 OutBack 복귀

### 보너스 카드 (최장도로/최강기사)
- **획득**: 화면 상단에서 오른쪽 핸드 끝으로 드로우
- **상실**: 핸드에서 위로 날아가며 사라짐 (다른 플레이어가 탈취 시)

---

## 카드 사용 플로우 (발전카드)

```
드래그 시작
  → 카드가 마우스를 따라감
  → 상단 임계선(화면 60% 높이) 초과?
    ├─ YES: 사용 가능 턴인가?
    │   ├─ YES → 카드 사용 확정, 위로 날아가며 효과 발동
    │   └─ NO  → 좌우 shake 연출, 핸드로 복귀
    └─ NO: 마우스 놓으면 핸드로 복귀
```

---

## 도적 디스카드 모드

```
7 초과 자원 보유 시 활성화
  → 왼쪽 핸드 "다중 선택 모드" 진입
  → 카드 클릭 시 토글 선택 (선택 시 위로 올라감)
  → 필요 디스카드 수 충족 → 확인 버튼 활성화
  → 확인 → 선택 카드들이 도적 위치로 날아감
  → 모드 해제
```

## 호버 확대 상세

| 항목 | 값 |
|------|-----|
| 확대 비율 | 1.3x ~ 1.5x |
| 위로 이동 | 카드 높이의 50% |
| 전환 시간 | 0.15s (OutBack) |
| 설명 표시 | 카드 상단에 툴팁 or 카드 자체 확대로 텍스트 가독 |
| 주변 카드 | 좌우로 살짝 밀려남 (공간 확보) |

## 구현 순서

### Phase 1 — 듀얼 핸드 기반
- [ ] CardHandManager를 2개 인스턴스로 운영 (Left/Right)
- [ ] 자원카드 데이터 클래스 추가 (ResourceCard)
- [ ] 타입별 정렬 로직 구현
- [ ] 호버 시 확대 + 위로 떠오르기 강화

### Phase 2 — 발전카드 사용 연출
- [ ] 드래그 → 상단 임계선 감지
- [ ] 사용 가능/불가 판정 연동 (IGameManager)
- [ ] 사용 불가 shake 연출
- [ ] 카드 사용 후 효과 발동 이벤트

### Phase 3 — 자원 카드 연동
- [ ] 타일 → 핸드 날아오기 연출 (월드→스크린→Canvas 좌표 변환)
- [ ] 건설/거래 시 카드 소모 연출
- [ ] 도적 디스카드 다중 선택 모드

### Phase 4 — 보너스 카드 + 폴리시
- [ ] 최장도로/최강기사 카드 (사용 불가 마크)
- [ ] 카드 타입별 색상/아이콘 비주얼
- [ ] WebGL 성능 최적화

## 기술 참고
- 기반: `Assets/Scripts/UI/CardHand/` (Balatro 프로토타입)
- UI: UGUI (Canvas) + DOTween
- 좌표 변환: `Camera.WorldToScreenPoint` → `RectTransformUtility.ScreenPointToLocalPointInRectangle`
- 데이터 소스: `PlayerState` (자원 배열), `DevelopmentCard` (발전카드 리스트)
