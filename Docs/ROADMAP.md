# ArcanaCatan → Arcana Board Module 로드맵

> 아르카나스톡의 보드게임 모듈. 카탄을 베이스로 경제 시뮬레이션 요소를 단계적으로 접목.

---

## Phase 1: 카탄 프로토타입 ✅

Claude Code + Unity 워크플로우 검증용 기본 카탄 구현.

- 헥스 그리드 (19타일, 큐브 좌표, pointy-top)
- 건설/거래/발전카드/도적 — 카탄 풀 룰셋
- AI 3난이도 (Easy/Medium/Hard) 대전
- UI Toolkit 기반 전체 UI (메뉴/로비/HUD/거래/결과)
- Netcode + Relay + Lobby 네트워크 기반 구축

---

## Phase 2: 제작 시스템 도입 🔄

카탄 자원(5종)을 원자재로, **가공품/완성품 제작 체인** 추가.
아르카나스톡의 핵심 경제 루프를 보드게임 위에서 실험.

**핵심 목표**:
- 원자재 → 가공품(1턴) → 완성품(3턴) 제작 파이프라인
- 제작소 건물 타입 + 배치 시스템
- 제작품 거래 (플레이어 간 + 은행)
- 승리 조건 확장 (제작 VP)

---

## Phase 3: 경제 시스템 확장

카탄의 고정 교환비율 → **동적 시장 경제**로 전환.

**핵심 목표**:
- 자원/제작품 시세 변동 (수요/공급 기반)
- 골드(G) 화폐 도입 — 자원 매매의 공통 단위
- 은행 → 상회 전환 (거래소 개념)
- 투자/배당 기초 시스템

---

## Phase 4: 시즌제 & 이벤트

아르카나스톡의 **듀얼 루프** 구조를 보드게임에 적용.

**핵심 목표**:
- 수확턴 (타일 주사위 2d6 → 보너스 수확)
- 이벤트턴 (섹터 주사위 2d6 → 시세 방향 결정)
- 시즌 3막 구조 (개척기/성장기/결산기)
- 시즌 보상 및 리셋 시스템

---

## Phase 5: 아트 & 비주얼

placeholder → 실제 아트 리소스 교체. 2.5D 비주얼 완성.

**핵심 목표**:
- 3D 타일/건물/캐릭터 모델
- 머티리얼/텍스처/라이팅
- 파티클 이펙트 (건설, 자원, 거래)
- BGM/SFX
- UI 스킨 (아르카나스톡 세계관 반영)

---

## Phase 6: 서버 & 인프라

P2P Relay → 서버 기반 아키텍처 전환.

**핵심 목표**:
- 서버 솔루션 선정 (TBD: PlayFab / 커스텀 / 기타)
- 계정 시스템 + 데이터 영속성
- 매칭 + 리더보드
- 호스트 마이그레이션 / 재접속

---

## Phase 7: 아르카나스톡 통합

독립 모듈 → 아르카나스톡 메인 프로젝트에 병합.

**핵심 목표**:
- 아르카나스톡 프로젝트로 코드 이전
- 본 게임 경제 시스템과 연동
- 보드게임 콘텐츠로서 인게임 삽입
- 공통 UI/UX 통일
