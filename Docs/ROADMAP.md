- [ ] 서버 솔루션 선정 (TBD: PlayFab / 커스텀 / 기타)
- [ ] 계정 시스템 + 데이터 영속성
- [ ] 매칭 + 리더보드

> ⚠️ **리스크 메모**: UGS 무료 티어 한도(CCU, Relay 대역폭) 카탄 단계에서 체감해볼 것. PlayFab/커스텀 서버 월 비용 구조 미리 조사. 도메인/SSL 세팅 경험도 필요
> 
> ⚠️ **QA 리스크**: 멀티플레이 테스트가 1인 개발 최대 병목. AI 자동 대전으로 어느 정도 커버 가능하나 한계 있음 — 실제 테스트 돌려보면서 자동화 범위 판단할 것. 로컬 테스트는 문제없으나 네트워크 QA 공수를 과소평가하지 말 것
