# ArcanaCatan - 온라인 카탄 게임 기획서

## 개요

| 항목 | 내용 |
|------|------|
| 장르 | 보드게임 (전략/자원관리) |
| 플랫폼 | WebGL (브라우저) |
| 시점 | 2.5D (탑다운 + 원근감) |
| 인원 | 3~4인 멀티플레이어 |
| 엔진 | Unity 6 (URP) |
| 네트워크 | Netcode for GameObjects + Relay + Lobby |
| UI | UI Toolkit |

## 핵심 게임플레이

카탄의 개척자들 규칙 기반. 자원을 모아 도로/마을/도시를 건설하고 **10승리점**을 먼저 달성한 플레이어가 승리.

### 자원 (5종)

| 자원 | 타일 | 용도 |
|------|------|------|
| 목재 | 숲 | 도로, 마을 |
| 벽돌 | 언덕 | 도로, 마을 |
| 양모 | 초원 | 마을, 발전카드 |
| 밀 | 밭 | 마을, 도시, 발전카드 |
| 광석 | 산 | 도시, 발전카드 |

### 건설 비용

| 건물 | 비용 | 승리점 |
|------|------|--------|
| 도로 | 목재 1 + 벽돌 1 | - |
| 마을 | 목재 1 + 벽돌 1 + 양모 1 + 밀 1 | 1점 |
| 도시 | 밀 2 + 광석 3 | 2점 |
| 발전카드 | 양모 1 + 밀 1 + 광석 1 | - |

### 턴 흐름

1. **주사위 굴림** — 해당 숫자 타일에 인접한 건물 소유자가 자원 획득
2. **거래** — 플레이어 간 거래 또는 항구/은행 거래 (4:1, 3:1, 2:1)
3. **건설** — 도로, 마을, 도시 건설 또는 발전카드 구매
4. **턴 종료**

### 특수 요소

- **도적** — 7 나올 시 도적 이동, 자원 8장 이상 보유자 절반 버림
- **발전카드** — 기사, 승리점, 독점, 풍년, 도로건설
- **최장교역로** — 도로 5개 이상 연속 (2점)
- **최대기사단** — 기사카드 3장 이상 사용 (2점)

## 보드 구성

- 헥스 타일 19개 (자원 5종 + 사막 1)
- 숫자 토큰 18개 (2~12, 7 제외)
- 항구 9개
- 교차점 54개 (마을/도시 배치)
- 변 72개 (도로 배치)

## 게임 흐름

```
메인 메뉴 → 로비 (방 생성/참가) → 대기실 → 초기 배치 → 본 게임 → 결과 화면
```

### 초기 배치

1. 순서대로 마을 1개 + 도로 1개 배치
2. 역순으로 마을 1개 + 도로 1개 배치
3. 두 번째 마을 인접 타일의 자원 획득

## 멀티플레이어

- **매칭**: Unity Lobby 서비스로 방 생성/검색/참가
- **연결**: Unity Relay로 P2P 연결 (서버 불필요)
- **동기화**: Netcode for GameObjects로 게임 상태 동기화
- **호스트**: 호스트-클라이언트 모델 (호스트가 게임 로직 권한)

## 씬 구성

| 씬 | 역할 |
|----|------|
| MainMenu | 메인 메뉴, 로그인 |
| Lobby | 방 목록, 생성/참가 |
| Game | 인게임 (보드, HUD, 거래) |

## UI 화면 목록

- 메인 메뉴
- 로비 (방 목록)
- 대기실 (플레이어 슬롯, 준비)
- 인게임 HUD (자원 표시, 주사위, 턴 정보)
- 거래창
- 건설 메뉴
- 발전카드 목록
- 결과 화면
